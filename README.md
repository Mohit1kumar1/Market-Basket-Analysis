# Market Basket Analysis in R

This repository contains an end-to-end market basket analysis conducted using R, focusing on understanding customer purchasing patterns from a retail store dataset. 
- Data: https://archive.ics.uci.edu/dataset/352/online+retail

## 📝 Dataset
The dataset contains information about:
- Customer transactions from a retail store.
- Key columns: InvoiceNo, Description (products), Quantity, and Country.
